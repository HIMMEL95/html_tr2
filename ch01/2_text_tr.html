여기서 부터는 눈에 보이는 영역으로 원하는 내용 기입

지금까지 배운 내용은 기본적으로 많이 사용되는 태그 관련 내용 학습

p 태그

p태그는 하나의 문단을 나타낼 때 많이 사용하는 태그이며, 최종 출력시에는 밑에 공백 한 줄이 추가된 상태로 출력된다.

hr 태그

hr태그는 화면상에 하나의 라인이 나올 수 있게 만들어주는 태그

br 태그

br 태그는 한 줄 띄어쓰기 할 때 사용하는 태그

span 태그

span태그는 Inline tag로 전체적인 영역을 잡는 것이 아니라, 일정 영역만 잡혀서 표현되는 것

blockquote 태그

blockquote 태그는 p태그와 비슷하게 작용하긴 하지만, 다른 점은 다른 글을 인용하거나 했을 때 인용된 글의 주위 영역에 공백을 주게 된다.

&nbsp;

&nbsp; 는 문장에서 2개 이상의 띄어쓰기를 나타낼 때 사용한다.

h 태그

h태그는 head tag로 글의 제목과 관련된 부분에 사용되게 된다. head tag의 종류는 1~6까지 있다.

속성

style

style 요소는 태그를 다르게 표현하기 위해서 적용하는 속성값이다.

bold

bold는 글에 굵기를 주고 싶을 때 사용

italic

italic은 글을 눕혀서 표현하고 싶을 때 사용

mark

mark는 글에 형광펜으로 표현되는 것처럼 표시 할 때 사용

small

small은 글을 작게 하고 싶을 때 사용

sub

sub는 아래첨자를 넣고 싶을 때 사용한다. H2O

sup

sup는 윗첨자를 넣고 싶을 때 사용한다. E=MC2

s

s는 잘못된 내용에 중간 밑줄을 치고 싶을 때 사용한다.

u

u는 글 아래에 밑줄을 넣고 싶을 때 사용한다.

abbr

abbr은 축약된 내용을 풀이할 때 사용한다. BH

q

q는 짧은 인용구를 인용할 때 사용한다. 청춘
